## Sobre o Projeto

Aplicativo Android de **Calculadora de Juros Simples** desenvolvido em **Kotlin**, criado como exercício prático do Capítulo 9 de um curso de desenvolvimento Android. O projeto foca no aprendizado de lógica de cálculo financeiro, manipulação de inputs do usuário e construção de interfaces com Android SDK.

---

## Tecnologias Utilizadas

- **Kotlin** — Linguagem principal do projeto
- **Android SDK** — Plataforma de desenvolvimento mobile
- **Gradle** — Gerenciamento de dependências e build

---

## Funcionalidades

- **Cálculo de Juros Simples** — Entrada de capital, taxa e período para cálculo automático
- **Exibição do Resultado** — Retorno do valor total com juros aplicados
- **Interface Simples e Direta** — Layout focado na usabilidade do cálculo

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/VryDeveloper/Calculadora-de-Juros.git
   ```
2. Abra o projeto no **Android Studio**
3. Aguarde a sincronização do Gradle
4. Execute em um emulador ou dispositivo físico com **Android 6.0+**
