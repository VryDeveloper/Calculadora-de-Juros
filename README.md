## About

Android **Simple Interest Calculator** app developed in **Kotlin**, built as a hands-on exercise from Chapter 9 of an Android development course. The project focuses on financial calculation logic, user input handling and building interfaces with the Android SDK.

---

## Technologies

- **Kotlin** — Main language
- **Android SDK** — Mobile development platform
- **Gradle** — Dependency management and build

---

## Features

- **Simple Interest Calculation** — Input capital, rate and period for automatic calculation
- **Result Display** — Returns the total amount with applied interest
- **Clean UI** — Layout focused on usability

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/VryDeveloper/Calculadora-de-Juros.git
   ```
2. Open the project in **Android Studio**
3. Wait for Gradle to sync
4. Run on an emulator or physical device with **Android 6.0+**
